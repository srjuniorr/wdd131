The Elden Beast Codex — WDD 131 Project

Site Name

The Elden Beast Codex

A codex is a reference tome or compendium, which fits a site that gathers lore and a battle guide in one place. The name also states the subject directly — the Elden Beast, the final boss of Elden Ring — so players searching for help recognise it instantly.

Site Purpose

The Elden Beast Codex is a fan reference site that helps Elden Ring players understand and defeat the game's final boss. It explains who the Elden Beast is, tells the story behind it, and provides a practical, phase-by-phase guide with an interactive, filterable list of the weapons that work best against it. A feedback page also lets players share and read each other's strategies.

Scenarios

These are questions a member of the target audience (Elden Ring players) might ask. The site's content is built to answer them:

  • "I keep dying to the Elden Beast — which weapons actually work against it?"
  • "I beat Radagon but I'm confused about the story. What exactly is the Elden Beast?"
  • "Which weapons should I avoid because the Beast resists them?"

Colour Scheme

The palette is "Gold & Cosmic Void" — a near-black background with golden accents, inspired by the Elden Beast drifting against a starry sky. The contrast keeps body text easy to read.

  • Void Black — #0d0e14
    Page background across the whole site.
  • Surface — #15161e
    Header, footer, and card backgrounds.
  • Parchment — #ece6d6
    Body text and paragraphs.
  • Order Gold — #d4af37
    Headings, links, buttons, borders, and active navigation.
  • Dark Amber — #8a6d3b
    Secondary accents, hover states, and dividers.

Typography

The site uses two Google Fonts. Headings use a display serif for an epic, engraved feel; body copy uses a classic serif chosen for readability.

Cinzel — The Elden Beast

Cinzel is used for the site title, all headings (h1–h3), the navigation brand, and button labels.

EB Garamond — The true final boss of Elden Ring is a divine vessel of the Greater Will.

EB Garamond is used for body text, paragraphs, lists, captions, and form fields.
